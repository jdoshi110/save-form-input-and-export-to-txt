Save-form-input-and-export-to-txt

Autosaves form input and then exports it to a .txt file when you're ready

    This will automatically save your user form input across sessions using local storage. You can export it into a .txt file when ready. I think it only really works in Chrome, but could probably be forced to work in other browsers.
